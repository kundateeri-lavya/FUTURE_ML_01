# FUTURE_ML_01
# 📈 Sales & Demand Forecasting for Businesses

## 📌 Project Overview

This project predicts future sales using historical sales data with Machine Learning (Linear Regression). It helps businesses estimate future demand and make better inventory and sales decisions.

---

## 🚀 Features

- Data Cleaning
- Feature Engineering
- Sales Forecasting using Linear Regression
- Future Sales Prediction
- Forecast Visualization
- CSV Export of Predictions

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- VS Code
